Warsh ERP - Client Install
==========================

For the customer:

1. Double-click:
   Setup-WarshERP.cmd

2. Choose the install folder.

3. Click Install.

4. After install, open the system from:
   - Desktop shortcut: Warsh ERP
   - Or Start Menu: Warsh ERP
   - Or the installed file: WarshERP.cmd

First run:

- The launcher opens a setup screen.
- Click Setup.
- Keep demo data checked if this is a demo installation.
- Click Open App.

Default login:

admin / admin123

Important:

- This version runs without Docker.
- It uses local SQLite database files inside the user's app data folder.
- The customer machine needs Windows PowerShell.
- If Electron runtime is not bundled in the package, the machine also needs Node.js/npm for first launch.
